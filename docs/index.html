The server can be accessed via the Secure Shell protocol for commandline use and file transfer. Popular clients are:

  • The ssh and scp commands on POSIX systems
  • File transfer tools like FileZilla (OSX, Windows, Linux) and WinSCP (Windows)
  • Terminal program PuTTY (Windows)...though, the Jupyter Notebook also provides a terminal

Users may add public keys to their .ssh/authorized_keys file. Do not remove the existing keys that are needed to utilize the client nodes.

The Jupyter Notebook supports Python 2, Python 3, and R. It also provides a terminal for command line use and simple file management.

hdfs:///user/molnar/data/election2012

This data-set contains over 170,000,000 tweets collected during 3 months leading up to the 2012 presidential elections.

The data-set is split up into files with about 1,000,000 to 10,000,000 status records in JSON format. The files are individually compressed with 'gzip'. Each individual file is valid. I suggest piping it through 'zcat' when reading in, rather than uncompressing the entire file.

Spark can read the compressed files directly.
